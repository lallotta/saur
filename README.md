# saur

Saur is my personal AUR helper for downloading PKGBUILDs and querying the AUR.

## Features

- Downloads PKGBUILDs using git
- Search the AUR
- Retrieve package information

## Usage

Search AUR:
```
saur search <search term(s)>
```

Print package information:
```
saur info <package(s)>
```

Download PKGBUILD:
```
saur get <package(s)>
```
